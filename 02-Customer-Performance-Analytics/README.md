# Customer Performance Analytics Dashboard (Power BI)

This dashboard analyzes customer sales performance with KPIs, trends, top customers, and regional breakdown.

## Key KPIs
- Total Sales
- Total Customers
- Total Transactions
- Average Sales per Customer

## Visuals Included
- Total Sales Over Time (trend)
- Total Sales by Region
- Top 10 Customers by Sales

- Screenshots (PNG)
- Exported report (PDF)
- Power BI file (PBIX) (if available)
